# rag_project

Small RAG (retrieval-augmented generation) starter scaffold.

Files created:
- `requirements.txt` — Python deps
- `.env` — environment variables
- `src/` — package with stub modules
- `tests/` — simple pytest placeholder

How to get started (Windows PowerShell):

1. Create a virtual environment and activate it:
   python -m venv .venv; .\.venv\Scripts\Activate.ps1
2. Install deps:
   pip install -r requirements.txt
3. Fill `.env` with `OPENAI_API_KEY` and any other settings.
4. Run tests:
   pytest -q
